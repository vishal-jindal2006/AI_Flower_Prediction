# 🌸 AI Flower Prediction System

A modern Machine Learning web application built using **Python**, **Streamlit**, and **Scikit-Learn** that predicts the species of an Iris flower based on its measurements.

---

## 🚀 Live Demo

🔗 https://aiflowerprediction.streamlit.app

---

## 📌 Features

✅ Real-time Flower Prediction
✅ Machine Learning Classification
✅ Confidence Score Display
✅ Interactive Dashboard
✅ Dataset Upload Support
✅ PDF Report Generation
✅ Modern Responsive UI
✅ Mobile Friendly Design
✅ Professional Dark Theme

## 🌸 Supported Flower Classes

The model predicts among:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

---

## 📊 Machine Learning Models

The application supports multiple ML algorithms:

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Support Vector Machine (SVM)
* Random Forest Classifier

The best-performing model is automatically selected and used for prediction.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Task-2-Vishal-Jindal.git
cd AI_Flower_Prediction
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Mac/Linux:

```bash
source venv/bin/activate
```

### Install Requirements

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run Prediction.py
```
